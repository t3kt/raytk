# simpleDiff

Category: combine
OP Type: raytk.operators.combine.simpleDiff



Combines two SDFs using the difference operator.
Produces the area of the first shape minus any areas overlapped by the second (or vice versa).

## Parameters

* `Enable`
* `Swaporder`
* `Inspect`
