# planeSdf

Category: sdf
OP Type: raytk.operators.sdf.planeSdf



An infinite plane on the x, y, or z axis.

* `Axis` - chooses which axis the plane faces.
* `Offset` - shifts the plane forwards or backwards along the axis that it faces.

## Parameters

* `Axis`
* `Offset`
* `Inspect`
