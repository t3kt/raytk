A vector field that evaluates to a constant color value.

This is the same as `constantField`, but the parameter is specified as a color instead of arbitrary float values.

## Parameters

* `Coordtype`
  * `float`
  * `vec2`
  * `vec3`
* `Contexttype`
  * `none`
  * `Context`
  * `MaterialContext`
  * `CameraContext`
  * `LightContext`
* `Color`
* `Alpha`
